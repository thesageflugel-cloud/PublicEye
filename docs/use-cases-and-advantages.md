# Use Cases and Advantages

## Primary Use Cases

### Citizen Use Cases

- Register an account and log in
- Submit a complaint with evidence image
- Provide ward, landmark, and optional GPS coordinates
- Track complaint status and SLA progress
- Add follow-up comments on an existing complaint

### Officer Use Cases

- View department complaint queue
- Update complaint status to `In Progress` or `Resolved`
- Upload proof of resolution
- Monitor escalated cases inside their department

### Admin Use Cases

- View all complaints across departments
- Assign complaints to officers and departments
- Create new officer and admin accounts
- Monitor department performance and complaint load
- Review overdue or escalated cases

## Advantages

- Improves transparency for both citizens and authorities
- Reduces dependency on informal complaint tracking
- Introduces measurable accountability through workflow stages
- Enables faster issue prioritization using priority and escalation levels
- Creates a permanent digital audit trail
- Provides a foundation for smart city or e-governance expansion

## Real-World Impact

- Faster pothole and drainage response
- Better garbage and sanitation handling
- Reduced ambiguity about complaint ownership
- Better public trust through visible status tracking
- Easier reporting for municipal performance reviews
